
# Lottery Number Generator

## Overview

A simple and visually appealing lottery number generator that provides a clean and modern user experience. The application will be responsive and work on both mobile and web.

## Design and Features

*   **Modern Aesthetics:** The application will have a visually balanced layout with clean spacing and polished styles. It will use a vibrant color palette, gradients, and drop shadows to create a sense of depth.
*   **Responsive Design:** The layout will adapt to different screen sizes, ensuring a seamless experience on all devices.
*   **Interactive Elements:** A prominent button will trigger the lottery number generation. The button will have a "glow" effect on interaction.
*   **Typography:** Expressive typography will be used to display the generated numbers, with a clear distinction between the main numbers and the bonus number.
*   **Number Generation:** The application will generate six unique random numbers from 1 to 45.

## Plan

1.  **HTML Structure (`index.html`):**
    *   Create a container for the lottery number generator.
    *   Add a title and a brief description.
    *   Create a display area for the generated numbers.
    *   Add a button to generate the numbers.

2.  **Styling (`style.css`):**
    *   Apply a modern and clean design with a vibrant color scheme.
    *   Use CSS variables for easy theming.
    *   Add responsive styles for different screen sizes.
    *   Style the number display and the generation button.

3.  **JavaScript Logic (`main.js`):**
    *   Implement the logic to generate six unique random numbers between 1 and 45.
    *   Create a function to update the DOM with the generated numbers.
    *   Add an event listener to the button to trigger the number generation.
